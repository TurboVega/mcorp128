Add with Carry (ADDC) Instruction

Purpose

This instruction adds two values, and stores the result into a register. The C (carry) bit is used as input to the addition, and will be output.

Usage

The following formats are supported for this instruction:

Format Encoding Operation Notes
addc Rd,#val 85H rdsel val
Rd ← Rd + val + C
 Value is 8-bits wide.
addc Rd,#val A5H rdsel val{7:0} val{15:8}
Rd ← Rd + val + C
 Value is 16-bits wide.
addc Rd,Rs C5H rdsel rssel
Rd ← Rd + Rs + C
  
addc Rd,Rs,#val E5H rdsel rssel val
Rd ← Rs + val + C
 Immediate value is as wide as Rs.

In this instruction, 'rdsel' selects the destination register, which is also the source register in the first two formats.

'rssel' selects the source register in the third and fourth formats.

Flags

The following flags are involved in this instruction:

15 14 13 12 11 10 09 08 07 06 05 04 03 02 01 00
- - NP GP FP IP HE SE GE FE IE V C N P Z
- - - - - - - - - - - * * * * *

Legend: 0 = cleared, 1 = set, ? = tested, * = updated, '-' = n/a
