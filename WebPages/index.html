MCORP128

My Custom Overlaid-Register Processor

  • 128 bytes of internal general-purpose register storage
  • instructions can process numbers up to 128 bits wide

 

These web pages describe the operation of my custom CPU, which can be implemented in HDL code (VHDL or Verilog), and synthesized into an FPGA. This is not a RISC CPU. It has some complex instructions, and instruction lengths (in bytes) vary.

The CPU has 128 byte-wide general-purpose registers. These are overlaid with 64 half-word-wide registers, 32 word-wide registers, 16 double-word-wide registers, and 8 quad-word-wide registers. Thus, the CPU can operate on 8-bit, 16-bit, 32-bit, 64-bit, or 128-bit numbers.

Details of the memory layout, I/O system, and interrupt handling are incomplete at present, and thus, TBD.
