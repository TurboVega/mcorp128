Branch Graphics Interrupt Enable Clear (BGC) Instruction

Purpose

This instruction branches to the specified program address, if the graphics interrupt enable (GE) bit is clear (i.e., if graphics interrupts are disabled).

Usage

The following formats are supported for this instruction:

Format Encoding Operation Notes
bgc label 36H offs{07:00}
if ¬ GE
  PC ← PC + offs
endif
 Offset is 8-bits wide.
bgc label 56H offs{07:00} offs{15:08}
if ¬ GE
  PC ← PC + offs
endif
 Offset is 16-bits wide.

Offset ('offs') is a signed value, and is sign-extended to 32-bits before being added to the PC.

Flags

The following flags are involved in this instruction:

15 14 13 12 11 10 09 08 07 06 05 04 03 02 01 00
- - NP GP FP IP HE SE GE FE IE V C N P Z
- - - - - - - - ? - - - - - - -

Legend: 0 = cleared, 1 = set, ? = tested, * = updated, '-' = n/a
