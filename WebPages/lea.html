Load Effective Address (LEA) Instruction

Purpose

This instruction loads the address of memory data into a register, in various ways.

Usage

The following formats are supported for this instruction:

Format Encoding Operation Notes
lea Rd,label 9CH rdsel offset
Rd ← PC + offset
 Offset is 8-bits wide.
lea Rd,label BCH rdsel offset{7:0} offset{15:8}
Rd ← PC + offset
 Offset is 16-bits wide.
lea Rd,[Rs] DCH rdsel rssel
Rd ← Rs
  
lea Rd,[Rs,#off] FCH rdsel rssel offset{7:0} offset{15:8}
Rd ← Rs + offset
 Offset is 16-bits wide.

In this instruction, 'rdsel' selects the destination register.

'rssel' selects the source register.

Flags

The following flags are involved in this instruction:

15 14 13 12 11 10 09 08 07 06 05 04 03 02 01 00
- - NP GP FP IP HE SE GE FE IE V C N P Z

Legend: 0 = cleared, 1 = set, ? = tested, * = updated, '-' = n/a
