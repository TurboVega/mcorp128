BSR/JSR Instruction

Purpose

This instruction calls a subroutine.

Usage

The following formats are supported for this instruction:

Format Encoding Operation Notes
bsr label 3AH offs{07:00}
SP ← SP-2; [SP] ← PC{31:16}
SP ← SP-2; [SP] ← PC{15:00}
PC ← PC + offs
 Offset is 8-bits wide.
bsr label 5AH offs{07:00} offs{15:08}
SP ← SP-2; [SP] ← PC{31:16}
SP ← SP-2; [SP] ← PC{15:00}
PC ← PC + offs
 Offset is 16-bits wide.
jsr Rs 79H rssel
SP ← SP-2; [SP] ← PC{31:16}
SP ← SP-2; [SP] ← PC{15:00}
PC ← Rs
 No offset is involved.
jsr [Rs] 7AH rssel
SP ← SP-2; [SP] ← PC{31:16}
SP ← SP-2; [SP] ← PC{15:00}
PC ← [Rs]
 Final location is indirect.

Offset ('offs') is a signed value, and is sign-extended to 32-bits before being added to the PC.

Register ('Rs', which is identified by 'rssel') must be a word (32-bit) register (RW00..RW31).
