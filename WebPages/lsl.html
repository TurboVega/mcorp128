Logical Shift Left (LSL) Instruction

Purpose

This instruction shifts a register to the left, replacing its lowest bit with a zero.

Usage

The following formats are supported for this instruction:

Format Encoding Operation
lsl Rd,#cnt 92H rdsel cnt
temp ← Rd
for i = 1 to cnt
  C ← temp{n-1}
  temp ← temp{n-2:0}, '0'
endfor
Rd ← temp
lsl Rd,Rs,#cnt F2H rdsel rssel cnt
temp ← Rs
for i = 1 to cnt
  C ← temp{n-1}
  temp ← temp{n-2:0}, '0'
endfor
Rd ← temp

In this instruction, 'rdsel' selects the destination register, which is the source register in the first format.

'rssel' selects the source register in the second format.

The number of preserved result bits depends on the width of the destination register.

Flags

The following flags are involved in this instruction:

15 14 13 12 11 10 09 08 07 06 05 04 03 02 01 00
- - NP GP FP IP HE SE GE FE IE V C N P Z
- - - - - - - - - - - * * * * *

Legend: 0 = cleared, 1 = set, ? = tested, * = updated, '-' = n/a
