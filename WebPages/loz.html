Low Zero Bit (LOZ) Instruction

Purpose

This instruction searches for the first bit with a value of zero (0), within the source register, starting with bit #0, and moving toward bit #n-1.

If it finds such a bit, then its bit number is stored in the lowest byte of the destination register, and the C (carry) bit is set, indicating successful location of a zero bit. Plus, one of 3 other flag bits is set, indicating where in the register the target bit was found.

If it does not find such a bit, then all 4 flag bits are cleared, indicating failure to locate a zero bit.

Usage

The following formats are supported for this instruction:

Format Encoding Operation Notes
loz Rd,Rs 67H rdsel rssel
C ← 0
Z ← 0
P ← 0
N ← 0
if zero bit is found
  C ← 1
  Rd{07:00} ← bit number
  if bit number == 0
    Z ← 1
  elseif bit number == n-1
    N ← 1
  else
    P ← 1
  endif
endif

In this instruction, 'rdsel' selects the destination register.

'rssel' selects the source register.
