CMP

Compare (CMP) Instruction

Purpose

This instruction subtracts one value from another, but does not store the result. The C (carry) bit is not used as (a borrow) input to the subtraction, but will be output.

Usage

The following formats are supported for this instruction:

Format Encoding Operation Notes
cmp Rd,#val 98H rdsel val
temp ← Rd - val
 Value is 8-bits wide.
cmp Rd,#val B8H rdsel val{7:0} val{15:8}
temp ← Rd - val
 Value is 16-bits wide.
cmp Rd,Rs C0H rdsel rssel
temp ← Rd - Rs
  

In this instruction, 'rdsel' selects the minuend register.

'rssel' selects the subtrahend in the third format.

Flags

The following flags are involved in this instruction:

15 14 13 12 11 10 09 08 07 06 05 04 03 02 01 00
- - NP GP FP IP HE SE GE FE IE V C N P Z
- - - - - - - - - - - * * * * *

Legend: 0 = cleared, 1 = set, ? = tested, * = updated, '-' = n/a
